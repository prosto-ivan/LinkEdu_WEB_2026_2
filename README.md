# 🎓 LinkEdu Hub — Адаптивний вебзастосунок для навчання

**LinkEdu Hub** — це сучасна платформа для каталогізації та відстеження освітніх ресурсів (відео, статей, курсів). Проєкт розроблений як лабораторна робота №1 з дисципліни «WEB-орієнтовані технології»

---

## 🌟 Основні можливості

- **Система ролей (RBAC):** Розмежування функціоналу для Гістя, Користувача та Адміністратора.
- **Особистий кабінет:** Вкладка «Моє навчання» для відстеження власного прогресу.
- **Динамічний контент:** Пошук у реальному часі та фільтрація за типом контенту (Відео/Статті/Курси).
- **Пагінація:** Зручна підгрузка карток за допомогою кнопки «Показати ще».
- **Історія відвідувань:** Клікабельний список останніх переглянутих матеріалів у сайдбарі.
- **Адаптивність:** Коректне відображення на смартфонах, планшетах та десктопах.
- **Темна тема:** Підтримка Dark Mode із збереженням вибору в `localStorage`.

---

## 🛠 Технологічний стек

- **HTML5:** Семантична верстка структури.
- **CSS3:** - **Grid Layout** для побудови основної сітки та футера.
  - **Flexbox** для вирівнювання елементів керування.
  - **CSS Variables** для реалізації темної теми.
- **JavaScript (Vanilla JS):** - Робота з DOM-деревом.
  - Фільтрація та маніпуляція масивами об'єктів.
  - Керування станом (State management) без зовнішніх бібліотек.
